## 概要

**Always CD Ripper** は、Windows向け高品質CDリッピングソフトです。  
音楽CDをFLAC / WAV形式で取り込み、AccurateRipデータベースとの照合により  
リッピングデータの正確性を検証します。  
MusicBrainz・GnuDBによる自動メタデータ取得とカバーアートの自動取得に対応したフリーソフトです。

---

## 主な特徴

- FLAC / WAV形式での高品質リッピング
- AccurateRip照合（リッピングデータの正確性を世界中のユーザーと照合）
- **Read Quality**表示（読み取り品質をリアルタイムで可視化）
- **Sector Map**（セクタごとの読み取り状態をカラーで表示）
- MusicBrainz / GnuDB による自動メタデータ取得
- カバーアート自動取得（MusicBrainz / Discogs）
- マルチディスク対応
- 読み取り速度設定（2x / 4x / 8x / 16x）
- サンプルオフセット補正

## インストール

同梱のインストーラー（MSIファイル）を実行してください。  
画面の指示に従って操作するとインストールが完了します。  
管理者権限が必要な場合があります。

---

## 使い方

1. CDをドライブに挿入し、ソフトを起動
2. **① Source** — CDドライブまたはCUE/BINファイルを選択
3. **② Output** — 出力先フォルダと出力形式（FLAC / WAV）を設定
4. **③ Metadata** — アルバム情報・曲名を確認・編集
5. **④ RIP** —「RIP START」ボタンをクリックしてリッピング開始
6. **⑤ Verify** — AccurateRipによる照合結果を確認

## Sector Mapの見方

リッピング中にセクタごとの読み取り状態を色で表示します。

| 色 | 意味 |
|---|---|
| 🔵 明るい青 | 正常に読み取れたセクタ |
| 🔵 暗い青 | やや読み取りが遅かったセクタ（軽微な傷の疑い） |
| 🟠 オレンジ | 読み取りが大幅に遅かったセクタ（傷の疑いが強い） |
| 🔴 赤 | 読み取りに失敗したセクタ |
| ⬛ 黒 | 未処理のセクタ |

新品のCDはほぼ全面が明るい青になります。  
傷のあるCDはオレンジや暗い青が点在します。

---

## Read Qualityについて

読み取り速度をもとにCDの状態を推定した品質指標です。

| Read Quality | 状態 |
|---|---|
| 100% | 全セクタ正常（新品同等） |
| 99%前後 | ごく軽微な傷あり（実用上問題なし） |
| 95%以下 | 傷が多い（AccurateRip照合で確認推奨） |

> **注意**: Read Quality はドライブの特性や読み取り速度設定により変動することがあります。参考値としてご活用ください。AccurateRip照合で「一致」が確認されれば、Read Qualityの値にかかわらずリッピングデータは正確です。

---

## AccurateRip照合について

リッピング完了後、Verifyステップで自動的にAccurateRipデータベースと照合します。  
世界中のユーザーが同じCDをリッピングしたデータと比較し、一致すれば「信頼度100%」と表示されます。

| 結果 | 意味 |
|---|---|
| ✦ 一致 | リッピングデータは正確です |
| ◈ 不一致 | CDの傷・ドライブの問題の可能性があります |
| — 未登録 | AccurateRipデータベースに未登録のCDです |

---

## 読み取り速度設定

Settings画面から読み取り速度を設定できます。

| 速度 | KB/s | 推奨用途 |
|---|---|---|
| 2x | 300 KB/s | 傷のあるCDに推奨（最も丁寧な読み取り） |
| 4x | 600 KB/s | バランス重視（デフォルト） |
| 8x | 1200 KB/s | 通常のCDに適した速度 |
| 16x | 2400 KB/s | 新品CDに限り推奨（高速リッピング） |

---

## サンプルオフセットについて

CDドライブごとの読み取り位置のズレを補正する設定です。  
ご使用のドライブのオフセット値は以下で確認できます。  
http://www.accuraterip.com/driveoffsets.htm

AccurateRip照合を重視する場合に設定してください。  
通常の用途では「0」のままで問題ありません。

## 対応ソース形式

- 物理CD（CDドライブ経由）
- CUE / BIN ファイル
- CUE / WAV ファイル

## 出力形式

- **FLAC**（可逆圧縮・メタデータ・カバーアート埋め込み対応）
- **WAV** （非圧縮・メタデータ・カバーアート埋め込み対応）
